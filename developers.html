Paiements électroniques pour société libre!

We are building Taler, a new electronic payment system for privacy and security.

Libre

Taler is free software implementing an open protocol. Anybody is welcome to inspect our code and integrate our reference implementation into their applications.

RESTful

Taler is designed to work on the Internet. Using a so-called RESTful protocol over HTTP or HTTPS, Taler is easy to integrate with existing Web applications.
